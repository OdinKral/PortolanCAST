Events anchor time.

A structured support or refutation attached to a hypothesis.

- Experiment
    
- External document
    
- Replication
    

Evidence is not narrative.  
It is traceable support.

### 7. Model

An integrated, structured explanation across components and hypotheses.

Models are living artifacts.  
They version.

### 8. Agent

A decision-making entity.

Examples:

- Technician
    
- Researcher
    
- AI system
    

Agents create hypotheses and observations.

      124 ## Epistemic Guardrails (Bias Prevention Architecture)
      125
      126 These are UX constraints, not just policies. They are embedded in the interface as intelligent friction
          .
      127
      128 **Rule 1 — Hypotheses cannot float**
      129 A Hypothesis node must reference ≥1 Observation and ≥1 Component or System. No free speculation nodes.
      130
      131 **Rule 2 — Evidence must be typed**
      132 Evidence must specify: Measurement · Experiment · Test · Document · Replication. Unspecified evidence i
          s disallowed.
      133
      134 **Rule 3 — Contradictions are first-class**
      135 Contradictions are equal visual weight to supporting evidence. No hiding disconfirming data. The system
           surfaces: *"No contradictory tests recorded for this hypothesis."*
      136
      137 **Rule 4 — Confidence is derived, not asserted**
      138 Confidence score derives from: (supporting evidence count) vs. (contradicting evidence count) + replica
          tion count. Prevents ego inflation.
      139
      140 **Rule 5 — Causation requires mechanism**
      141 To draw a `causes` edge, the user must provide a one-sentence mechanism description and at least one ev
          idence reference. Slows sloppy causal inference.
      142
      143 **Rule 6 — Versioning is mandatory**
      144 Models cannot be overwritten. Model v1 → `version_of` → Model v2. The system rewards updating correctly
          , not being right.
